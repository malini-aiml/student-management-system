# Student Management System

A Python-based command-line project to manage student records.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- Input Validation
- Duplicate Roll Number Check
- JSON File Storage

## Technologies Used
- Python
- JSON
- File Handling

## How to Run

1. Install Python
2. Run:

python main.py
